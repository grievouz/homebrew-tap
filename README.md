# homebrew-tap

A [Homebrew tap](https://docs.brew.sh/Taps) hosting formulas for [grievouz](https://github.com/grievouz)'s tools.

## Formulas

### git-wt

Git worktree helper. Source: [grievouz/git-wt](https://github.com/grievouz/git-wt).

```sh
brew install grievouz/tap/git-wt
```
